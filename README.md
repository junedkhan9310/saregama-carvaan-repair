# Carvaan MP3 Repair & Conversion

## TL;DR

Repair and convert corrupted **Saregama Carvaan** MP3 files to standard MP3 format so they play on any device. Works entirely in **Google Colab**.

---

## My Story

My mom had some favorite songs saved on a **Saregama Carvaan** memory card. The songs were **not playing properly in the Carvaan itself**, so I opened the memory card and tried to play them on my laptop.

Some files **still wouldn’t play correctly**, probably due to a **bug in the Carvaan or old memory issues**, so I wasn’t exactly sure why they weren’t working.

I didn’t want her music to be lost, so I worked on **repairing the files using this script**. It took some effort, trial and error, and learning along the way, but thanks to AI, I was able to pull it off and **convert them into normal MP3 format**.

This is my little project that I posted on GitHub — it might help someone else with the same problem.

This notebook lets anyone **repair and convert Carvaan music files easily**, so the songs can play again on any device.

---

## About the Project

This is a **Google Colab notebook** that helps you:

* Repair corrupted MP3 files from Carvaan memory cards
* Convert all files to **standard MP3 format** (128 kbps, stereo, 44.1 kHz)
* Zip the final files for easy download

It works entirely in **Google Colab**, so you don’t need to install anything on your computer.

---

## How to Use

1. Open the notebook in [Google Colab](https://colab.research.google.com/).
2. Upload your Carvaan MP3 files.
3. Run all the cells step by step.
4. The notebook will repair, convert, and zip your files.
5. Download the `.zip` containing all playable MP3s.

---

## Requirements

* Google Colab (no setup needed)
* ffmpeg (already included in Colab)
* Your own Carvaan MP3 files
